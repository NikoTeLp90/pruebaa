Dentro del vercel.json configurar:

{
  "version": 2,
  "builds": [
    {
      "src": "proyecto/wsgi.py",
      "use": "@vercel/python",
      "config": { "maxLambdaSize": "15mb" }
    }
  ],
  "routes": [
    {
      "src": "/(.*)",
      "dest": "proyecto/wsgi.py"
    }
  ]
}


dentro del settings.py modificar:

ALLOWED_HOSTS = ['.vercel.app', 'localhost', '127.0.0.1']
